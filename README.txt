# AmicuServices - Eco-Conscious Travel Platform

## Quick Start
Run server: cd server && npm install && npm start
Then open index.html in your browser

## Services
- AmicusTravel - Sustainable travel recommendations
- AmicusDocs - Document authentication and legalization services
- AmicusC02 - Carbon footprint reporting for businesses
- AmicusTransport - Low-carbon airport transfers
- AmicusFitness - Fitness classes search (Coming Soon)

## Main Pages
- index.html - Homepage
- amicusservices-landing.html - Services overview
- amicustravel-landing.html - Travel services
- amicusdocs-landing.html - Document services
- amicusc02-landing.html - Carbon reporting
- amicustransport-landing.html - Transport options
- sustainable-hotel-search.html - Hotel search
- lca-calculator.html - Carbon calculator

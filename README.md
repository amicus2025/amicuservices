## ğŸš€ Getting Started

### Prerequisites
- Node.js and npm installed
- A modern web browser

### Installation

1. Clone the repository:
```bash
git clone https://github.com/amicus2025/amicuservices.git
cd amicuservices
```

2. Install server dependencies:
```bash
cd server
npm install
```

3. Start the server:
```bash
npm start
```

4. Open `index.html` in your web browser or use a local development server
